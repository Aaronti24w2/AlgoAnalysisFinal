Question 1: What Type of search is efficient on a sorted array?

Answer: B = Binary Search.

Question 2: What is the time complexity of inserting a new node at the
beginning of a singly linked list?

Answer: A = O(1)

Question 3: In a binary search tree (BST), the left child of a node is always:

Answer: B = Smaller than the node.

Question 4: Which traversal of a BST visits nodes in ascending order?

Answer: C = In-Order Traversal

Question 5: In a doubly linked list, what does the prev pointer in the first
node point to?

Answer: B = Null
